# Command Line Packager
This program creates a package called `data01.arc`.

## Build
* On Linux:
```
make pack-linux
```
* On macOS:
```
make pack-mac
```
* On WSL2/MSYS2 with mingw-w64:
```
make pack.exe
```

## Run
Put a pack binary to a game directory and run it.

## Result
`data01.arc` will be created.
